# Minha Loja - Site de E-commerce

Este projeto é um site de e-commerce simples construído com HTML, CSS e JavaScript.

## Estrutura do Projeto

### Loja (Loja)
O diretório `loja/` contém o site principal de e-commerce com as seguintes páginas:
- `bem-vindo.html` - Página de boas-vindas com botões de login e cadastro
- `cadastro/signup.html` - Página de registro de usuário
- `login/login.html` - Página de login do usuário
- `sucssful-login/index.html` - Página de sucesso após cadastro
- `cookies/cookie.html` - Página de consentimento de cookies
- `loading-anim/loading.html` - Página de animação de carregamento

### Funcionalidades da Loja
- Design responsivo com Tailwind CSS
- Páginas de autenticação de usuário (login/cadastro)
- Animações de carregamento
- Banner de consentimento de cookies
- Suporte a favicon para todas as páginas
- Rodapé profissional com links e redes sociais



## Tecnologias Utilizadas

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Ícones Font Awesome

## Instalação e Configuração

### Pré-requisitos
- Um servidor web (para as páginas HTML, ou abra diretamente no navegador)

### Executando a Loja (Páginas HTML)
1. Navegue até o diretório `loja/`
2. Abra `bem-vindo.html` no seu navegador web
3. Navegue pelas páginas usando os botões
